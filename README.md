To-Do List App

A simple web-based To-Do List application built using HTML, CSS, and JavaScript.
This project was created as part of a mini software engineering assignment.

Features

- Add task with date
- Delete task
- Filter tasks (All / Today)
- Input validation
- Modern UI with gradient design
- Responsive layout
- Interactive filter buttons with active state

Tech Stack

- HTML5
- CSS3 (Flexbox, Gradient, Animation)
- Vanilla JavaScript (DOM Manipulation)

How It Works

- Enter a task and select a date.
- Click Add to insert the task into the list.
- Use All to display all tasks.
- Use Today to filter tasks based on the current date.
- Click Delete to remove a task from the list.

Purpose
This project demonstrates fundamental frontend development skills, including:
